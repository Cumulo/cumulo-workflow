
Cumulo Worflow
------

> Cumulo apps project template.

Features:

* hot swapping client
* hot swapping server
* declarative programming

Based on [Calcit Editor](https://github.com/Cirru/calcit-editor).

### Usages

Start developing app:

```bash
cd app/
yarn
calcit-editor
# with another terminal
yarn watch
# with another terminal
yarn dev
# open http://localhost:7000
```

Start developing server:

```bash
cd server/
yarn
calcit-editor
# another terminal
yarn watch
# another terminal
node target/main.js
```

### Build

Build client:

```bash
cd app/
yarn build
```

Build server:

```bash
cd server/
yarn build
```
